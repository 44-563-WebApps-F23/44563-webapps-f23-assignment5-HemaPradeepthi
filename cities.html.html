The row containing the city names will break first when reducing the browser size.
This is because the city names are in larger columns compared to the single letters in the first row. In Bootstrap, the responsive grid system causes the larger columns to stack vertically on smaller screens, so the city names will wrap onto multiple lines earlier than the single letters.
